# Games 1.0.0

Première release publique de Games, une collection de jeux classiques jouables sur mobile et directement dans le navigateur.

Tester la version web :

```text
https://ronylicha.github.io/games/
```

## Jeux inclus

### Dames

- Jeu complet avec moteur de coups.
- Captures, promotion en dame et coups légaux.
- Mode 1 vs IA ou 1 vs 1 local.
- Choix de la couleur du joueur.
- Dernier mouvement IA marqué visuellement.
- Sauvegarde automatique de la partie.

### Échecs

- Moteur basé sur `chess.js`.
- Vrais pions PNG dédiés.
- Choix de la couleur du joueur.
- Niveaux d'IA facile, normal et difficile.
- Mode 1 vs IA ou 1 vs 1 local.
- Dernier coup IA mis en évidence.
- Sauvegarde automatique de la partie.

### Backgammon

- Plateau mobile optimisé et plateau horizontal sur desktop web.
- Lancers de dés, coups légaux et zones de sortie.
- Pions capturés empilés au centre.
- Mode 1 vs IA ou 1 vs 1 local.
- Dernier coup IA marqué.
- Sauvegarde automatique.

### Dominos

- Jeu de dominos contre l'IA.
- Tuiles classiques en PNG.
- Pioche et coups légaux.
- Plateau en serpent lisible sur mobile.
- Lignes de 5 tuiles avec connecteurs verticaux.

### Solitaire

- Klondike solo.
- Option pioche infinie ou limitée à 3 passages.
- Fondations, tableau, réserve et défausse.
- Sauvegarde automatique tant qu'une nouvelle partie n'est pas lancée.

### Dino Run

- Runner arcade avec cactus, oiseaux, nuages et score.
- Alternance jour/nuit tous les 250 points.
- Leaderboard top 5 persistant.
- Modale custom pour saisir le nom du joueur en fin de partie.
- Sur desktop web, flèche haut pour sauter et flèche bas pour se baisser.

## Plateformes

- Web : GitHub Pages.
- Android : APK preview via EAS Build.
- iOS : compatible Expo/EAS.
- Local : Expo Go, émulateur Android, simulateur iOS ou navigateur.

## Technique

- Expo SDK 56.
- React Native 0.85.
- React 19.
- Expo Router.
- TypeScript.
- `expo-image` pour les assets PNG.
- `@react-native-async-storage/async-storage` pour les sauvegardes locales.
- GitHub Actions pour le déploiement web.
