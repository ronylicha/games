# Contributing to Games

Thanks for contributing to Games.

## License Rules

- Code, documentation, tests, scripts, and configuration are accepted under GPL-3.0-only.
- Original game and branding assets are proprietary under [ASSETS-LICENSE.md](./ASSETS-LICENSE.md).
- The project name, logo, package name, and visual identity are governed by [TRADEMARKS.md](./TRADEMARKS.md).
- Contributions require acceptance of [CLA.md](./CLA.md).

## Pull Request CLA Confirmation

Every pull request must include this sentence in the PR body:

```text
I have read and agree to the Games Contributor License Agreement.
```

If you are contributing on behalf of an employer or other organization, confirm that you are authorized to submit the contribution.

## Asset Contributions

Do not submit third-party assets unless their source and license are clearly documented.

Do not submit original assets unless you agree they may be governed by [ASSETS-LICENSE.md](./ASSETS-LICENSE.md), or unless a separate written agreement is made before merge.

## Forks

Forks are welcome under the GPL-3.0-only code license. Public releases of forks must replace proprietary assets and branding, and must use a different package name than `com.ronylicha.games`.
